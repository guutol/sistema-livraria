# Sistema Livraria

Sistema de gerenciamento de livraria desenvolvido em Java para fins de estudo.

## Sobre o projeto

Este projeto simula o funcionamento básico de uma livraria/biblioteca em ambiente de console, permitindo o cadastro de livros e clientes, além da realização de empréstimos e listagem dos dados cadastrados.

Foi desenvolvido com foco em praticar conceitos fundamentais de Programação Orientada a Objetos, organização de classes, manipulação de listas e interação com o usuário via terminal.

## Funcionalidades

- Cadastrar livros
- Cadastrar clientes
- Realizar empréstimo de livros por nome ou ID
- Listar livros cadastrados
- Listar clientes cadastrados
- Verificar disponibilidade dos livros

## Estrutura do projeto

O sistema é composto pelas seguintes classes:

- `Main` → responsável pela execução do programa e menu interativo
- `Biblioteca` → concentra as regras principais do sistema
- `Livro` → representa os livros cadastrados
- `Autor` → representa os autores dos livros
- `Cliente` → representa os clientes cadastrados
- `Emprestimo` → representa os empréstimos realizados

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos
- IntelliJ IDEA
